# 美术管线

> 本项目的美术资产生产流程。工具在 `tools/pixel-pipeline/`，资产在 `assets/art/`。  
> 详细规则见 `tools/pixel-pipeline/PIXEL_RULES.md`；AI 工具见 `tools/pixel-pipeline/AIGEN_TOOL.md`。

---

## 分轨原则（B.8）

| 轨道 | 适用范围 | 工具 |
|---|---|---|
| **像素轨** | 游戏世界（tile/角色/物品/装备图标） | Pillow（Python），SunshineFlow AI |
| **古风矢量轨** | UI/界面（水墨/卷轴/古风框） | SVG/HTML-CSS |

两轨不混。像素用 `tools/pixel-pipeline/`，UI 用独立矢量工具。

---

## 程序化 vs AI vs 手绘的边界

| 类别 | 谁做 | 状态 |
|---|---|---|
| 地形 tile / 无缝纹理 / 噪声地表 | 纯 code（numpy+Pillow） | 待实现 |
| autotile / spritesheet 打包 | 纯 code | 待实现 |
| 装备/法宝图标（静态） | **AI 网格批量**（SunshineFlow）+ code 切片/派生 | ✅ 管线就绪 |
| 功法图标（静态） | AI（SunshineFlow image2，带 alpha） | ✅ 可出 |
| 角色立绘/图鉴（静态） | AI（SunshineFlow pixel-art-fs） | ✅ 可出 |
| 角色本体（可动画） | **程序化骨架**（char_skeleton.py，过渡占位） | ✅ 占位可用，Unity 阶段走 Spine+美术 |
| 角色装备叠层（武器/法宝） | AI 出静态件 + 挂骨架手/背挂点 | ✅ 管线就绪 |
| 精细原创立绘/流畅表情动画 | ❌ AI/真美术，code 做不到 | Unity 阶段 |

**角色本体不用 AI 出**——AI 逐帧骨架会漂移，动画就烂了。Unity 阶段走 Spine/DragonBones。

## AI 出料流程（装备/图标/立绘）

```
1. 取 prompt  → tools/pixel-pipeline/AI_PROMPTS.md（逐部件已写好）
2. 生图        → /sunshineflow-banana-alpha（SunshineFlow API）
               → image2 蓝图: 带 alpha 透明通道（图标/立绘用）
               → pixel-art-fs 蓝图: 像素游戏资产（角色/sprite sheet）
3. 省额度技巧  → 一次出 4×4 网格图（16件/次）
               → postprocess.py grid 切片 → 独立 48² PNG
4. 品阶派生   → equip_variants.py（1件→凡/灵/仙三品阶）
5. 后处理      → postprocess.py（高清→游戏尺寸）
6. 入库        → assets/art/pixel/parts/（精修后）
```

## 尺寸纪律

| 资源类型 | 尺寸 | 说明 |
|---|---|---|
| 装备/法宝/功法图标 | 48² px | 游戏内尺寸，AI 出高清后 postprocess 缩 |
| 角色本体 | 32×48 px | char_skeleton.py 基准，骨架驱动 |
| 地形 tile | 32² px | 必须无缝可平铺 |

AI 直出约 1254² RGBA，只作"出图中转"——`postprocess.py` 负责缩到目标格。高清不入库。

## 快速开始

```bash
cd E:/Seal/jiuye

# 出装备网格（4×4=16件）
python .claude/skills/sunshineflow-banana-alpha/scripts/generate_image.py \
  --prompt "pixel art equipment icons, a 4x4 grid of 16 different..." \
  --output-dir "./tools/pixel-pipeline/_aigen"

# 切片成独立件
python tools/pixel-pipeline/postprocess.py grid <网格图> \
  --rows 4 --cols 4 --cell 48 \
  --names jian,dao,... --out-dir tools/pixel-pipeline/_aigen/sliced

# 品阶派生（1件→3品阶）
python tools/pixel-pipeline/equip_variants.py <件.png> --out-dir <output>

# 生成角色（骨架占位）
python tools/pixel-pipeline/char_gen.py
python tools/pixel-pipeline/char_skeleton.py
```
