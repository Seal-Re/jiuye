# 模块化效果系统 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 21 路战斗机制从 `Note` 字符串升为结构化整数模块，让当前版本所有角色在 R2 战斗里真参战（非空壳），全整数确定、不破 off 逐字节、防 MVP 蒙混。

**Architecture:** 升级 `EffectOp` 为模块原语（末尾加可选 `Amount2`/`Trigger`/`Rarity`，向后兼容几百处 4 参调用）；稀有档 9 新 `EffectOpKind`（参数化复用）+ 唯一档 `Special` 派发 per-path 纯整数 handler（受控逃逸口）；`DuelEngine.ResolveR2` 三回合同时结算应用模块；功法门控防御（Evade 连续缩放/护体）；无 RNG。

**Tech Stack:** C#/.NET（`Jianghu.Core` netstandard2.1），xUnit，确定性整数，IL 浮点扫描（`ILFloatScanner`）。

**上游设计（权威）:** `docs/legacy-specs/specs/2026-06-14-v1.2-B5-模块化效果系统-design.md`（**§15 复验补丁覆盖前文冲突处**）+ R2 战斗稿 §1-13。

**实现顺序:** 批1(框架)→批2(普通/稀有迁)→批3(Special)→批4(R2引擎)→批5(法宝)→批6(硬化gate)。批1 是全员依赖，先行。

**红线:** `Jianghu.Cultivation` 禁浮点（IL 扫描守）；off 逐字节不变（off 无 Cultivation 进不了 R2）；新增语义算子标 `// L1`；Special handler 不读 daoHeart/innerDemon、不消费随机、副作用经 chokepoint。

```csharp
// CombatContext.cs — 不暴露裸 Dict(§15.3)
public enum Side { Attacker, Defender }
public sealed class CombatContext {
    private readonly CultivationState _atk, _def;
    public int ReadResource(Side s, string key) => (s==Side.Attacker?_atk:_def).Resources.TryGetValue(key, out var v)?v:0;
    public void ApplyResource(Side s, string key, int delta) => (s==Side.Attacker?_atk:_def).ApplyResource(key, delta); // 经钳
    public bool HasTag(Side s, string tag) => /* 读 path SituationalTags + flags */;
    // ... ctor 注入双方 state/path/stat
}
// ModuleResolver.cs
public static int ApplyOnUse(int dmg, EffectOp m, CombatContext ctx) => m.Kind switch {
    EffectOpKind.AddPenInteger   => dmg + m.Amount,
    EffectOpKind.PenFromResource => dmg + ctx.ReadResource(Side.Attacker, m.Key!) * m.Amount / Den(m),
    EffectOpKind.AoePerTarget    => dmg + m.Amount,                       // R2单挑×1
    EffectOpKind.CounterMul      => ctx.HasTag(Side.Defender, m.Key!) ? CapCounter(dmg, m) : dmg,
    EffectOpKind.DrainResource   => DrainAndReturn(dmg, m, ctx),
    EffectOpKind.Backlash        => dmg, // 自伤在 selfDmg 通道, 见批4
    _ => dmg,                                                             // Dot/Control 标记在批4
};
static int Den(EffectOp m) => m.Amount2 >= 1 ? m.Amount2 : 1;            // §15.6 钳
static int CapCounter(int dmg, EffectOp m) => System.Math.Min(dmg * m.Amount / Den(m), dmg * 3 / 2); // §15.4 联合上界×3/2
```
